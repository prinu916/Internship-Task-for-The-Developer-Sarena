# E-commerce Sales Analysis Report

## Objective

The goal of this project was to analyze e-commerce sales data and identify important business insights using Python.

---

## Tools Used

- Python
- Pandas
- Matplotlib

---

## Analysis Performed

- Data loading
- Data cleaning
- Revenue calculation
- Product-wise analysis
- Monthly sales trend analysis
- Data visualization

---

## Visualizations

### 1. Bar Chart
Shows comparison of sales across products.

### 2. Line Chart
Shows monthly sales trends over time.

---

## Insights

- Laptop products generated the highest revenue.
- Sales performance improved over multiple months.
- Visualizations helped identify trends clearly.

---

## Conclusion

This project demonstrates a complete data analysis workflow including loading, cleaning, analyzing, and visualizing data using Python.
