# Sales Data Analysis Report

## Project Overview

This project analyzes a simple sales dataset using Python and Pandas.

The analysis includes:
- Loading CSV data
- Exploring dataset information
- Cleaning missing values
- Removing duplicate records
- Calculating sales metrics
- Creating a formatted report

---

## Metrics Calculated

1. Total Revenue
2. Best-Selling Product
3. Highest Sale
4. Average Sales

---

## Results

- Total Revenue was calculated using the Total_Sales column.
- The best-selling product was identified by grouping sales by product name.
- Missing values were handled using fillna().
- Duplicate rows were removed using drop_duplicates().

---

## Technologies Used

- Python 3
- Pandas Library
