# Personal Introduction Program

A beginner-friendly Python project that collects user information and displays a personalized welcome message using Python basics such as variables, input handling, and formatted output.

---

## 📌 Project Overview

This project was created as part of a beginner Python internship task.  
The program interacts with the user by asking simple questions and then generates a friendly introduction message.

The goal of this project is to practice:
- Python syntax
- Variables
- User input
- String formatting
- Basic program flow

---

## 🚀 Features

- Takes user input using `input()`
- Stores data in variables
- Uses formatted strings (`f-strings`)
- Displays a personalized welcome message
- Beginner-friendly and easy to understand

---

## 🛠️ Technologies Used

- Python 3

No external libraries are required.
