# Student Grade Calculator

A beginner-friendly Python project that takes a student's name and marks, then returns a grade with an encouraging message.

## Features
- Uses `if-elif-else` statements for grading logic
- Validates marks input so only values from 0 to 100 are accepted
- Uses a `while` loop to keep asking until valid input is entered
- Includes reusable functions for clean code
- Shows encouraging messages for every grade

## Grading Logic
- A: 90-100
- B: 80-89
- C: 70-79
- D: 60-69
- F: 0-59
